## 🧐  <span style="margin-right: 10px"></span> What's inside?

This starter kit focuses on simplicity. You will get main infrastructure about your project, and you will create your own rules. You can change directory names, you can add new libraries and you can also create a magic 🪄

## Getting Started

### 📦 <span style="margin-right: 10px"></span> Prerequisites

- Node (v16.10.0+)

- Npm (v7.24.0+)

<br />

### ⚙️ <span style="margin-right: 10px"></span>  How To Use

```
$ git clone https://github.com/itsemirhanengin/react-starter.git my-awesome-project
```

```
$ cd my-awesome-project
```

```
$ yarn install
```

```
$ yarn start
```

### 🚀  <span style="margin-right: 10px"></span> To publish your app

```
$ yarn build
```
